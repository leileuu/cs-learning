## 📚 学习内容

### Day 1-2: 变量与数据类型
```python
# 变量
name = "小明"
age = 18
height = 1.75
is_student = True

# 打印
print(f"我叫{name}，今年{age}岁")
```

### Day 3-4: 条件语句与循环
```python
# if 条件
if age >= 18:
    print("成年人")
else:
    print("未成年")

# for 循环
for i in range(5):
    print(i)

# while 循环
count = 0
while count < 5:
    print(count)
    count += 1
```

### Day 5-6: 函数
```python
def greet(name):
    return f"你好，{name}！"

result = greet("小明")
print(result)
```
