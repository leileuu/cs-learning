## 🗓️ 建议学习计划

### 第一个月：Python 基础
- 第1-2周：Python 语法、数据类型、函数
- 第3周：面向对象、文件操作
- 第4周：做一个小工具（爬虫/计算器）

### 第二个月：前端基础
- 第1-2周：HTML + CSS，写静态页面
- 第3周：JavaScript 基础
- 第4周：做一个 Todo App

### 第三个月：前端框架
- 第1-2周：React / Vue 框架
- 第3周：组件化开发
- 第4周：接一个公开 API 做项目

### 第四个月：后端入门
- 第1-2周：Node.js / Python Flask
- 第3周：数据库 MySQL
- 第4周：写一个 REST API

### 第五-六个月：全栈项目
- 整合前后端技能
- Docker 部署
- 完成一个完整全栈项目
