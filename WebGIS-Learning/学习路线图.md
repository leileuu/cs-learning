# 🗺️ WebGIS 开发学习路线图

> 8 周从零到 WebGIS 入门

---

## 📊 整体时间规划

```
Week 1-2:   前端基础（HTML/CSS/JS）
Week 3-4:   Leaflet 地图库入门
Week 5-6:   GeoJSON 数据与可视化
Week 7-8:   完整项目实战
```

---

## ✅ Week 1-2: 前端基础

### 学习目标
- 掌握 HTML 页面结构
- 掌握 CSS 布局（Flexbox）
- 掌握 JavaScript 基础

### 每日任务

**Day 1-3: HTML**
- [ ] HTML 标签（div, span, p, h1-h6）
- [ ] 表单元素（input, button）
- [ ] 链接和图片

**Day 4-7: CSS**
- [ ] 选择器（class, id, 标签）
- [ ] Flexbox 布局
- [ ] 响应式设计基础
- [ ] 做一个静态网页

**Day 8-14: JavaScript**
- [ ] 变量和数据类型
- [ ] 函数和作用域
- [ ] DOM 操作（getElementById, querySelector）
- [ ] 事件监听（addEventListener）
- [ ] JSON 解析

### 练习项目
- [ ] 个人介绍网页
- [ ] Todo List 应用

---

## ✅ Week 3-4: Leaflet 地图库入门

### 学习目标
- 在网页中嵌入地图
- 添加标记和交互

### 每日任务

**Day 15-17: 第一个地图**
- [ ] CDN 引入 Leaflet
- [ ] 创建地图容器（div#map）
- [ ] 初始化地图（设置中心点、缩放级别）
- [ ] 添加底图（OpenStreetMap）

**Day 18-20: 地图元素**
- [ ] 添加标记（Marker）
- [ ] 添加折线（Polyline）
- [ ] 添加多边形（Polygon）
- [ ] 添加弹窗（Popup）

**Day 21-24: 图层与控制**
- [ ] 图层组（LayerGroup）
- [ ] 图层控制（Control.Layers）
- [ ] 自定义图标
- [ ] 事件处理（click, mouseover）

**Day 25-28: 进阶**
- [ ] 定位（GeoLocation API）
- [ ] 测量距离
- [ ] 绘制工具（Leaflet.draw）

### 练习项目
- [ ] 标记你的城市地图
- [ ] 添加多个地点的旅游攻略地图

---

## ✅ Week 5-6: GeoJSON 数据与可视化

### 学习目标
- 理解 GeoJSON 格式
- 加载和显示地理数据
- 数据可视化（热力图、分级填色图）

### 每日任务

**Day 29-32: GeoJSON 基础**
- [ ] GeoJSON 格式讲解
- [ ] Point（点）数据
- [ ] LineString（线）数据
- [ ] Polygon（面）数据
- [ ] FeatureCollection

**Day 33-36: 数据加载**
- [ ] L.geoJSON 加载
- [ ] onEachFeature 交互
- [ ] style 样式函数
- [ ] pointToLayer 自定义图标

**Day 37-42: 数据可视化**
- [ ] Choropleth Map（分级填色图）
- [ ] Heatmap（热力图）插件
- [ ] 自定义 Legend（图例）
- [ ] Info Box（信息框）

### 练习项目
- [ ] 中国主要城市分布图
- [ ] 各省人口分级填色图

---

## ✅ Week 7-8: 完整项目实战

### 项目：校园地图应用

**Day 43-50: 开发**
- [ ] 设计数据库
- [ ] 创建 PostgreSQL + PostGIS 表
- [ ] 编写后端 API
- [ ] 前端页面搭建
- [ ] 地图功能实现
- [ ] 数据接入

**Day 51-56: 功能完善**
- [ ] 搜索功能
- [ ] 路线规划（Leaflet Routing Machine）
- [ ] 图层切换
- [ ] 响应式适配

**Day 57-60: 部署上线**
- [ ] GitHub 仓库
- [ ] 前端部署（Vercel）
- [ ] 后端部署（Railway/Render）
- [ ] 测试上线
